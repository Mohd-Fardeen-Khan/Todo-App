# 📝 To-Do App

A clean and modern To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add tasks
- Delete tasks
- Responsive UI
- Modern glassmorphism design

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript (DOM Manipulation)
